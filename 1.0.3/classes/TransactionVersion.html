Class TransactionVersion

Static class containing transaction version constants.

Transaction format versions are defined in catapult-server in each transaction's plugin source code.

In catapult-server, the DEFINE_TRANSACTION_CONSTANTS macro is used to define the TYPE and VERSION of the transaction format.

see

https://github.com/nemtech/catapult-server/blob/main/plugins/txes/transfer/src/model/TransferTransaction.h#L37
